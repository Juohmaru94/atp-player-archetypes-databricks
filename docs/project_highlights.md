# Project Highlights

## What this project does
This project analyzes ATP tennis data from 2020 to 2024 to identify player archetypes, study match statistics associated with winning, and generate player-level performance case studies.

## Strongest technical components
- normalized player-match table design
- player-season aggregation
- clustering-ready feature engineering
- KMeans archetype modeling
- descriptive win-rate bucket analysis
- player comparison analysis
- chart pack for portfolio presentation

## Strongest analytical findings
- first-serve points won percentage is very strongly associated with winning
- second-serve points won percentage is also strongly associated with winning
- aces are positively associated with winning
- double faults are negatively associated with winning
- grass shows the strongest average serve effectiveness in the sample
- players in the same archetype can still differ significantly by surface profile

## Best portfolio takeaway
This project shows the ability to go from raw sports data to cleaned analytical tables, player segmentation, insight generation, and portfolio-ready visual storytelling.
