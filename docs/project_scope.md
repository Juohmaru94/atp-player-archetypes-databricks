# Project Scope

## Project Title
ATP Player Archetypes & Performance Insights in Databricks

## Goal
Build a Databricks-based tennis analytics portfolio project using ATP historical match data to:
- engineer analytics-ready data tables
- cluster players into style archetypes
- analyze match statistics associated with winning
- produce player-level performance insight dashboards and case studies

## Initial Scope
- ATP singles matches
- Main tour data
- Historical analysis using 1991–2024 data
- Bronze / Silver / Gold architecture
- SQL + PySpark transformations
- Clustering as the main analytical method
- Descriptive performance-driver analysis
- BI/dashboard outputs

## Out of Scope for V1
- Live data ingestion
- Real-time predictions
- Betting recommendations
- Challenger/ITF expansion
- Deep causal inference
- Production deployment outside portfolio/demo context
