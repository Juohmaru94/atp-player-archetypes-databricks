# Results Summary

## Archetype Modeling

The project clustered ATP player-seasons from 2020 to 2024 into four first-pass archetypes:

- High-Risk Big Server
- Balanced All-Court Player
- Lower-Power Grinder
- Elite Big Server

These archetypes were built from player-season performance features including:
- win rate
- average ranking
- aces per match
- double faults per match
- first serve in rate
- first serve win rate
- second serve win rate
- break points saved rate

The clustering work shows that ATP players can be grouped into interpretable style categories, while still preserving meaningful player-to-player differences within the same broad cluster.

## Key Descriptive Insights

### 1. First serve in percentage is positively associated with winning
Win rate increased from 35.8% in the below-50% bucket to 61.8% in the 75–80% bucket.

### 2. First serve points won percentage is very strongly associated with winning
Win rate increased from 3.2% in the below-55% bucket to 90.4% in the 85%+ bucket.

### 3. Second serve points won percentage is also strongly associated with winning
Win rate increased from 15.2% in the below-40% bucket to 92.3% in the 70%+ bucket.

### 4. Aces are positively associated with winning
Win rate rose from 32.9% in the 0–1 ace bucket to 63.4% in the 16+ ace bucket.

### 5. Double faults are negatively associated with winning
Win rate declined from 59.2% in the 0–1 double-fault bucket to around 38–40% in the highest buckets.

### 6. Surface matters
Grass had the highest average first-serve win rate and ace production, while clay had the lowest ace volume and lowest first-serve effectiveness.

## Player Case Studies

### Stefanos Tsitsipas
- 345 matches
- 70.1% win rate
- classified as Elite Big Server in every season from 2020 to 2024
- strongest surface in this sample: clay

### Daniil Medvedev
- 330 matches
- 75.5% win rate
- classified as Elite Big Server in every season from 2020 to 2024
- strongest surface in this sample: hard court

## Comparison Takeaway

Tsitsipas and Medvedev were both classified in the same broad archetype, but their surface strengths differed meaningfully:
- Tsitsipas performed best on clay
- Medvedev performed best on hard court

This is an important portfolio result because it shows that clustering is useful as a high-level summary, but not as a complete replacement for player-specific analysis.

## Important Caveat

All insight work in this project is descriptive. The analysis identifies associations in historical ATP match data, but it does not prove causation.
